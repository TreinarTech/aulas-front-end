Inputs

Os inputs são elementos fundamentais em formulários HTML, permitindo a coleta de dados do usuário.

Tipos de Inputs

  • text: Campo de texto simples.
  • password: Campo para senhas, oculta os caracteres digitados.
  • email: Campo para endereços de e-mail, valida o formato.
  • number: Campo para números, permite definir um intervalo.
  • checkbox: Caixa de seleção, permite múltiplas escolhas.
  • radio: Botões de opção, permite uma única escolha entre várias opções.
  • submit: Botão para enviar o formulário.
  • reset: Botão para limpar os campos do formulário.
  • file: Campo para upload de arquivos.
  • date: Campo para seleção de datas.
  • color: Campo para seleção de cores.
