Estrutura de semântica

HTML5 introduziu elementos semânticos que melhoram a acessibilidade e a compreensão do conteúdo.

Elementos Semânticos

  • header: Define o cabeçalho de uma página ou seção.
  • nav: Define uma seção de navegação.
  • main: Define o conteúdo principal do documento.
  • article: Define um conteúdo independente e autocontido.
  • section: Define uma seção genérica de conteúdo.
  • aside: Define conteúdo relacionado, mas não essencial ao conteúdo principal.
  • footer: Define o rodapé de uma página ou seção.
  • figure: Define conteúdo ilustrativo, como imagens ou diagramas.
  • figcaption: Define uma legenda para o elemento figure.
  • time: Define uma data ou hora.
  • mark: Define texto destacado.
  • progress: Define o progresso de uma tarefa.
  • meter: Define uma medição dentro de um intervalo conhecido
